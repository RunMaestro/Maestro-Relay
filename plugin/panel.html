Maestro Relay

Bridge Discord & Slack bots to your Maestro agents.

This panel is write-only: it sends changes to the plugin and cannot read state back. Every result — saved, connected, bound, or an error — appears as a Maestro toast, not here. Token fields are blank on load; leave a token empty to keep the one already stored.
